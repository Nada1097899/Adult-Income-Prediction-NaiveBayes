# Adult Income Prediction using Naive Bayes

A machine learning project aimed at predicting whether an individual's annual income exceeds $50,000 based on census data (age, education, occupation, etc.).

##  Project Overview
This project explores the "Adult Census Income" dataset to build a classification model. It involves significant data cleaning, handling missing values, and applying probabilistic modeling to achieve accurate predictions.

## Tech Stack
* **Language:** Python
* **Machine Learning:** Gaussian Naive Bayes
* **Data Preprocessing:** RobustScaler, Category Encoders
* **Libraries:** Pandas, NumPy, Scikit-learn, Seaborn, Matplotlib.

## Key Technical Highlights
- **Data Cleaning:** Identified and handled missing values (encoded as '?') in 'workclass', 'occupation', and 'native-country'.
- **Feature Scaling:** Applied `RobustScaler` to numerical features to minimize the influence of outliers.
- **Probabilistic Modeling:** Implemented the Gaussian Naive Bayes classifier, achieving high performance on the test set.
- **Evaluation:** Used Confusion Matrix and Classification Reports to validate model robustness.

## Key Results
- **Model Accuracy:** Strong performance in classifying income brackets.
- **Insights:** Identified key factors like education level and age that significantly impact income earning potential.
